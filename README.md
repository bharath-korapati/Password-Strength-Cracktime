# Password-Strength-Cracktime

A legitimate, defensive password awareness web app.

✅ Strength score via zxcvbn (client-side)  
✅ Crack-time estimates across realistic scenarios  
✅ Improvement tips + safer alternative generation  
✅ Dark mode  
✅ Learn / Privacy / About pages  

## Ethics & Safety
This tool does NOT:
- brute-force logins
- attempt account access
- perform credential stuffing or hacking

It provides *estimates* to encourage stronger passwords and safer habits (unique passwords + MFA/passkeys).  
Avoid pasting real passwords used on important accounts.
